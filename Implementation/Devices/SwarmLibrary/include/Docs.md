# include

## Purpose

Top-level project include directory for the SwarmLibrary PlatformIO project.
Currently empty; this is a placeholder for any application-level private headers.

Public library headers are located in lib/swarm_library/include/ and are
exported automatically when other projects reference SwarmLibrary via lib_deps.
