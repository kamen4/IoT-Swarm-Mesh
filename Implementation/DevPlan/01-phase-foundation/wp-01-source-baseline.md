# WP-01 Source Baseline

## Goal

Lock canonical source references and normalize interpretation boundaries.

## Tasks

T1. Build canonical source inventory
- T1.1 Enumerate all files in Protocol/_docs_v1.0 used by implementation.
- T1.2 Mark each file as normative or contextual.
- T1.3 Link each source file to domain area.

T2. Build exclusion inventory
- T2.1 List simulation-only files not used as normative requirements.
- T2.2 List research artifacts considered advisory only.

T3. Produce source baseline report
- T3.1 Document inclusion criteria.
- T3.2 Document exclusion criteria.
- T3.3 Obtain review and approval.

## Deliverables

- Source baseline document.
- Normative vs contextual classification matrix.

## Acceptance criteria

- Every implementation requirement can point to at least one normative source file.
