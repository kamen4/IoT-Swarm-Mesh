A command packet sent from the HubDevice to a specific EmitterDevice to set its on/off state.

The hub generates ControlPacket instances automatically at the interval configured on each EmitterDevice via ControlFrequencyTicks. When an EmitterDevice receives this packet it toggles its State to Command.
