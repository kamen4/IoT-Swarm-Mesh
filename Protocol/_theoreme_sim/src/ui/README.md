# ui

UI composition modules for controls and simulation metrics.

## Files

- dom.js - Small DOM creation helpers.
- controlPanel.js - Inputs and action buttons, including batch research controls and progress popup.
- metricsPanel.js - Status rendering for assumptions/lemmas/metrics.
- infoModal.js - Header info modal open/close and accessibility behavior.
- liveRangeValue.js - Live slider value synchronization helpers.
- collapsibleGroups.js - Shared details-group state helpers for collapsible panels.

## Research batch note

- Research batch export is HTML-only.
- JSON and CSV are downloaded from buttons inside the generated HTML report.
