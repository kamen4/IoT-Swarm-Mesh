# RBAC and Users

## Role Types

- User
- Dedicated Admin
- Admin

## Required Access Principles

- User capabilities are restricted to regular operational actions.
- Dedicated Admin can manage devices and users within assigned authority.
- Admin can assign and revoke dedicated admin privileges.

## Wave 1 Policy

- Role checks MUST be enforced in HUB-facing operations.
- Privilege changes MUST be traceable.
