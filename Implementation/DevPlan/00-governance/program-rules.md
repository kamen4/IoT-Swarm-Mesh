# Program Rules

## Objective

Define mandatory delivery rules for the full implementation program.

## Rules

1. Traceability rule
- Every requirement in implementation tasks must map to source docs.

2. Open-decision rule
- No implementation starts on areas marked OPEN DECISION without owner and deadline.

3. Scope-control rule
- No undocumented protocol features are introduced in implementation phases.

4. Phase-gate rule
- Phase exit only when all gate criteria are satisfied.

5. Evidence rule
- Each completed work package must produce concrete artifacts and verification evidence.

## Mandatory planning artifacts

- Requirement register.
- Decision register.
- Risk register.
- Dependency register.
- Gate checklist per phase.
