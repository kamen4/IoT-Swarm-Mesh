# WebApp/WebApp.Client/Layout

Contains Blazor layout components that define the outer visual shell of the application.

---

## Files

| File               | Responsibility                                                                                                                                                                                |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MainLayout.razor` | The default layout for all pages. Renders the `@Body` content region. Currently a minimal wrapper; the application top bar and navigation controls are rendered directly inside `Home.razor`. |
