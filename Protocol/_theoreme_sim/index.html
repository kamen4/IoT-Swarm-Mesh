Charge-Induced Tree Theorem Simulator

IoT Swarm Mesh: mathematical DOWN-tree simulation

Theorem and simulation guide

Theorem summary

The simulator demonstrates that strict charge increase toward the gateway induces an acyclic parent graph that reaches the root on the forward-eligible subset.

  • A5: gateway charge is the unique strict maximum.
  • A6: each eligible non-root node has an eligible neighbor with higher charge.
  • A7: parent is selected among neighbors with strictly higher charge.

How the algorithm works

  1. Generate a connected topology and initialize node charges.
  2. Propagate neighbor estimates by max update over link deliveries.
  3. Select parents with strict charge monotonicity and tie-break penalties.
  4. Rebuild parent and children maps and evaluate theorem checks.
  5. Optionally simulate DOWN broadcast along eligible child edges.

How to read the simulation

  • Node color encodes charge level; gateway is emphasized.
  • Link thickness and brightness encode observed weight values.
  • Tree links are highlighted, with dq labels showing parent-child charge deltas.
  • Metrics panel reports assumptions, lemma checks, and theorem pass/fail.
  • Broadcast report shows traversal order and duplicate detection status.
