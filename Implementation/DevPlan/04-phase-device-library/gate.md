# Phase 4 Gate

## Gate checklist

- Library API and callback contracts approved.
- Onboarding and security behavior approved.
- IO and event behavior approved.
- Routing and sleepy-device behavior approved.

## Evidence required

- API contract verification evidence.
- Onboarding/security verification evidence.
- IO/event verification evidence.
- Routing/sleep behavior verification evidence.

## Gate outcome

- Pass: device library stream is integration-ready.
- Fail: unresolved issues return to corresponding device-library work package.
