Class PacketRouter

Namespace
Engine.Routers
Assembly
Engine.dll

Implements a flooding broadcast routing strategy. When asked to route a packet from a given device, the router queries SimulationEngine for all devices visible to that device and enqueues a cloned copy of the packet addressed to each of them. This is the sole routing mechanism currently used by the engine.
