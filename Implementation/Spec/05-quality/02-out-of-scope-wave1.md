# Out of Scope (Wave 1)

This file lists explicit exclusions for Wave 1 execution.

## Exclusions

- Multi-gateway root coordination.
- End-to-end immediate command delivery guarantee for deep-sleep devices.
- Automatic online parameter optimization in production (PROVISIONAL policy exclusion).
- New confidentiality mechanisms beyond documented authenticity model.
- Protocol redesign outside documented message registry and routing model.

## Notes

- Exclusions can be reconsidered in later waves after Wave 1 quality gates pass.
- Any scope extension MUST be added to source map and devplan dependency map first.
