Namespace Engine.Statistics

Classes

SimulationStatistics

A singleton that listens to SimulationEngine events and accumulates simulation metrics.

Extensibility: add a new StatMetric to Metrics and subscribe to any SimulationEngine event to populate it. The Updated event is raised after every change so UI layers can re-render reactively without polling.

Every tick a TickSnapshot is appended to History (capped at MaxHistoryLength entries) for time-series charts.

StatMetric

A single observable metric tracked by SimulationStatistics. UI layers bind to DisplayValue and listen to Updated for re-renders.

TickSnapshot

An immutable snapshot of all plottable metric values captured at the end of a single engine tick. Stored in the SimulationStatistics history ring-buffer and consumed by the chart UI.
