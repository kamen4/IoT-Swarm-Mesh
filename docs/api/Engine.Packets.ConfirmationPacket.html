Class ConfirmationPacket

Namespace
Engine.Packets
Assembly
Engine.dll

A special packet that is automatically generated and sent back to the original sender when a packet with NeedConfirmation set to true is successfully delivered. The payload contains the Guid bytes of the confirmed packet, allowing the sender to correlate the acknowledgement with the original transmission.
