Class Device

Namespace
Engine.Devices
Assembly
Engine.dll

Abstract base class for all devices in the simulation network. A device has a unique identity and a position in 2D space. When it receives a packet it either forwards it (if it is not the intended recipient) or accepts and processes it.

Forwarding is delegated to RoutePacket(Packet, Device) so that the active IPacketRouter is always consulted. Devices never reference a concrete router directly.
