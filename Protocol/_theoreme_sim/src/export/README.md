# export

Result export utilities for headless simulation mode.

## Files

- snapshotBuilder.js - Builds compact per-round and final snapshots for export.
- jsonExport.js - Generates JSON payload from headless simulation result.
- csvExport.js - Generates CSV table from per-round snapshots.
- download.js - Browser-safe text file download helper.
- resultsExporter.js - Orchestrates JSON + CSV export in one action.
