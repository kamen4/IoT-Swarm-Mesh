Class HubDevice

Namespace
Engine.Devices
Assembly
Engine.dll

Represents the central hub (gateway) device of the simulation network. The hub is the designated destination for packets generated by GeneratorDevice instances and the root of the network topology. It is registered as the singleton hub inside SimulationEngine when added to the simulation.

The hub also emits protocol control traffic:

  • BEACON messages to seed DOWN-tree parent selection.
  • DECAY messages to prevent unbounded charge growth.
